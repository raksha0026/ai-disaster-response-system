# AI Disaster Response & Resource Management System

## Overview
This project is a Python-based console application that helps manage disaster incidents by allowing users to report disasters, view incidents, and allocate emergency resources.

## Features
- Report disaster incidents
- View reported incidents
- Allocate emergency resources
- Track incident status

## Technologies Used
- Python

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/raksha0026/ai-disaster-response-system.git
   ```

2. Open the project folder.

3. Run the application:
   ```bash
   python app.py
   ```
